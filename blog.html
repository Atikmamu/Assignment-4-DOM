What is the Document Object Model (DOM)?

The DOM is a programming interface that represents the structure of an HTML or XML document as a tree of objects. It allows developers to access, modify, and manipulate the content, structure, and style of a webpage dynamically using programming languages like JavaScript, enabling interactive web experiences.

How do you select an element from the DOM?

In JavaScript, elements are selected from the DOM using methods like document.getElementById() to access an element by its ID, document.querySelector() for selecting via CSS selectors, or document.getElementsByClassName() for accessing elements by class name, enabling dynamic manipulation of webpage content.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a single event listener is attached to a parent element to manage events from its child elements. Instead of adding multiple listeners, the parent handles events through bubbling. This improves performance and simplifies dynamic content handling in the DOM.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes in the DOM, JavaScript methods like setAttribute() and getAttribute() are used. For styles, properties can be modified via the style object, e.g., element.style.color = 'red'. This allows dynamic updates to an element's appearance and behavior.
